# 问题一：项目说明

## 背景与题意对应

本题对应赛题 **问题 1**：仅使用**班组 1**的设备，完成 **A 车间**全部整修工序；目标为总完工时间（Makespan）尽量短，并输出每台设备在各工序上的起止时刻与持续作业时间（秒，向上取整），即表 1 所列字段。

详细题意拆解见：`src/Qbase/competitionB.md` 第六节「问题一逐句解析」。

## 工程结构

- 求解脚本：`src/Q1/solve_q1.py`
- 公共模块：`src/Qbase/`（数据加载、解析、时间计算、调度、校验、导出）
- 输入数据：`code/data/` 下三份 Excel
- 输出结果：`src/Q1/outputs/`（表 1、校验报告）

## 如何运行

1. 安装依赖：`pip install -r code/requirements.txt`
2. 将官方「工序流程表」「班组配置表」「车间距离表」放入 `code/data/`（文件名与上一致）。
3. 执行：`python code/src/Q1/solve_q1.py`（工作目录为 `code`）。

## 模型与算法说明（简述）

- **工序**：车间内按编号顺序执行；含双设备时两类设备各完成全部工程量，工序完成时间为两类设备完工时间的最大值。
- **设备**：班组 1 内同类型多台设备可并行服务不同工序，但同一台设备在时间上不可重叠。
- **运输**：设备从「班组 1」或某车间前往另一车间时计入 `ceil(距离/速度)` 秒；同一车间内相邻工序不计运输。
- **求解**：采用**贪心事件推进**策略分配设备，快速得到满足全部约束的可行解及 Makespan；如需理论最优可在此基础上再建整数规划或 CP-SAT 模型。

## 输出说明

- `表1_问题1调度结果.xlsx`：工作表「表1_调度明细」为设备级记录；「汇总」给出完成问题 1 的最短时长（秒）（当前为算法所得可行解对应值）。
- `validation_report.txt`：对工序顺序、重复展开、类型匹配、设备互斥、作业时长字段及运输约束的自动检查摘要。

## 依赖

- Python 3.10+
- `pandas`、`openpyxl`
