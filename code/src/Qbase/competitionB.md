| 车间距离表 | 班组、车间之间的距离信息 | 运输时间计算参数 |
| 时间起点 | 00:00:00 | 调度起始时间 |
| 时间精度 | 精确到秒且向上取整 | 离散时间计算规则 |
| 预算 | 500000元 | 问题4预算约束 |

### 10.4 约束条件（场景规则 / 资源限制）

| 约束名称 | 原文依据 | 约束含义 |
|---|---|---|
| 工序顺序固定 | 各车间内工序顺序固定，即所有工序必须严格按照给定顺序依次执行 | 同一车间内后一工序必须在前一工序完成后执行 |
| 双设备共同完成 | 两类设备都需要完成该工序所对应的工程量，两类设备都完成后工序才完成 | 多设备任务完成时间取各设备完成时间后的完成判定 |
| 设备数量固定 | 各班组内设备数量固定 | 问题1-3不允许新增设备 |
| 设备可复用 | 设备在不同工序作业之间可重复使用 | 同一设备可执行多道工序 |
| 设备互斥 | 同一时刻每台设备只能服务于一道工序作业 | 同一设备任务时间不能重叠 |
| 同车间转移忽略 | 同一台设备在同一车间的不同工序之间使用时，设备转移运输时间忽略 | 同车间连续作业运输时间为0 |
| 跨车间转移不可忽略 | 同一台设备在不同车间使用时，跨车间设备转移运输时间不可忽略 | 跨车间连续作业需计入运输时间 |
| 预算约束 | 企业拟追加总额为500000元的设备购置预算 | 购买设备总费用不得超过500000元 |

## 11. 关键逻辑链标注

### 11.1 数据字段到分析目标的逻辑链

| 数据来源 | 数据字段 | 中间计算 / 约束 | 分析目标 |
|---|---|---|---|
| 工序流程表 | 工序编号、所属车间、先后关系 | 构建每个车间的工序链 | 保证工序顺序执行 |
| 工序流程表 | 工程量、作业效率 | 计算设备持续工作时间 | 得到每条设备任务的作业时长 |
| 工序流程表 | 所需设备类型 | 匹配可用设备 | 确定设备分配方案 |

### 11.2 决策变量到约束的逻辑链

| 决策内容 | 对应约束 | 原文依据 |
|---|---|---|
| 某道工序由哪台设备执行 | 设备类型匹配；设备互斥 | 各工序需占用特定类型设备完成；同一时刻每台设备只能服务于一道工序 |
| 同一设备执行多个工序的顺序 | 跨车间运输时间；同车间运输时间忽略 | 同一车间转移运输时间忽略；不同车间转移运输时间不可忽略 |
| 某车间下一道工序何时开始 | 工序顺序固定 | 各车间内工序顺序固定 |
| 双设备工序何时完成 | 两类设备均完成后才完成 | 两类设备都完成该工序对应工程量后，工序才可判定完成 |
| 问题4购买多少设备 | 预算约束；设备资源扩容 | 500000元设备购置预算；各班组补增设备资源 |

### 11.3 参数依赖逻辑链

| 参数 | 依赖关系 | 用途 |
|---|---|---|
| 作业持续时间 | 工程量 ÷ 作业效率，并按秒向上取整 | 确定设备任务时间长度 |
| 运输时间 | 车间距离 ÷ 设备移动速度，并按秒向上取整 | 确定设备跨车间转移时间 |
| 工序完成时间 | 该工序所需设备任务完成时间的完成判定 | 确定后续工序开始条件 |
| 总完成时间 | 所有工序完成时间中的最大值 | 各问题最短时长目标 |
| 购买总费用 | 设备单价 × 购买台数 | 问题4预算约束 |

## 12. 核心遵循原理汇总

| 原理类型 | 对应题目内容 | 说明 |
|---|---|---|
| 优化逻辑 | 计算完成任务的最短时长 | 以完成时间最短为优化目标 |
| 调度逻辑 | 多工序、设备、车间之间的安排 | 需要决定工序执行时间与设备分配 |
| 资源约束逻辑 | 设备类型、设备编号、设备数量、同一时刻只能服务一道工序 | 设备是有限资源，必须满足互斥与类型匹配 |
| 优先关系逻辑 | 工序按编号由小到大完成 | 后续工序依赖前序工序完成 |
| 同步完成逻辑 | 双设备工序需两类设备都完成工程量后才完成 | 工序完成时间由多个设备任务共同决定 |
| 运输时间逻辑 | 跨车间设备转运时间不可忽略 | 设备任务顺序与空间位置影响调度时间 |
| 成本预算逻辑 | 问题4中500000元设备购置预算 | 设备购买方案受预算限制 |
| 离散时间逻辑 | 起始时间00:00:00，精确到秒且向上取整 | 所有时间计算应采用秒级离散时间 |

## 14. 不得改变的原文题意边界

| 边界 | 原文依据 | 说明 |
|---|---|---|
| 车间范围固定 | A、B、C、D、E 五个车间 | 不应加入其他车间 |
| 工序顺序固定 | 按照编号数字由小到大完成 | 不应改变车间内部工序顺序 |
| 双设备完成规则固定 | 两类设备都完成工程量后工序才完成 | 不应将双设备工序简化为单设备工序 |
| 设备互斥固定 | 同一时刻每台设备只能服务于一道工序 | 不应允许设备并行服务多个工序 |
| 跨车间运输时间必须考虑 | 跨车间设备转移运输时间不可忽略 | 不应忽略不同车间间转运时间 |
| 同车间运输时间忽略 | 同一车间不同工序之间使用时运输时间忽略 | 不应给同车间连续作业额外增加运输时间 |
| 问题4预算固定 | 总额为500000元 | 不应超过预算 |
| 时间精度固定 | 精确到秒且向上取整 | 不应使用非秒级时间或向下取整 |
